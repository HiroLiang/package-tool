Systex 包板工具
===

Systex 包板工具，可進行 java8. 11. 17 包板工作。
---

## 開發規格：

1. JDK : 17
2. framework : Spring Boot
3. frontend : Vue
4. server : Tomcat
5. runtime env : Docker - alpine

---

## 修改所需知識點

1. Spring Boot - 伺服器框架
2. Web Socket - 包板指令通信 ( 簡易通信，不需 Stomp, Sock.js 技術 )
3. Java runtime, Process 物件使用 - 與 server 環境溝通
4. Vue - 前端框架
5. Docker - 部署環境配置
